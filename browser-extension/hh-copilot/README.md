# OfferGO HH Copilot

Chrome/Edge extension для подготовки индивидуальных откликов на hh.ru без hh API и без WPF.

## Установка для разработки

1. Откройте `chrome://extensions` или `edge://extensions`.
2. Включите `Developer mode`.
3. Нажмите `Load unpacked`.
4. Выберите папку `browser-extension/hh-copilot`.
5. Откройте OfferGO: `/cover-materials/auto-responses`.
6. Создайте код подключения и введите его в popup расширения.

## Поведение

- До подключения popup показывает только поле кода и кнопку.
- После подключения расширение получает список резюме напрямую из OfferGO API.
- На странице вакансии hh.ru расширение считывает текст, генерирует отклик и вставляет письмо в форму.
- Финальная отправка отклика не автоматизирована: пользователь проверяет форму и нажимает отправку сам.
