# Billing

## Current state

Покупка подписки временно отключена до подключения нового банка.

Что работает сейчас:

- тарифы и лимиты хранятся в БД;
- активные entitlements продолжают учитываться;
- история существующих платежей доступна пользователю;
- quota enforcement не зависит от платежного провайдера.

Что отключено:

- создание платежной ссылки;
- внешний webhook платежного провайдера;
- синхронизация pending-платежей с внешним провайдером.

## User-facing behavior

При попытке купить подписку интерфейс должен показывать:

> Функция покупки подписки пока недоступна.

## Ownership

- billing domain: `packages/billing`
- billing API: `apps/api/src/billing`
- billing UI: `apps/web/app/(dashboard)/billing`

Новый банк нужно подключать через отдельный adapter внутри `packages/billing`, не размазывая провайдерскую логику по web/API.
